🍎 AquilLife: Health-Retail & Analytical System
Project Live on GitHub
AquilLife is a full-stack e-commerce platform integrated with a dedicated Business Intelligence (BI) Dashboard. It bridges the gap between retail operations (PHP/Oracle XE) and data-driven decision-making (Power BI).

📊 Key Insights & Analytics
This project isn't just a shop; it's a data tool. The integrated Power BI dashboard tracks:
Sales Trends: Monthly revenue growth and peak sales periods.
Product Performance: Identification of top-selling nutrition supplements.
Customer Demographics: Analysis of buying patterns to optimize inventory.

🏗️ Database Architecture
The backend is powered by a relational Oracle database. I designed the schema to ensure Data Integrity and optimized query performance for the analytics layer.
Key Tables:
AQUIL_USERS: Manages customer profiles and authentication.
AQUIL_PRODUCTS: Tracks inventory, categories, and pricing.
AQUIL_ORDERS: Records transaction history for the BI pipeline.

🚀 Features
Full-Stack Retail: Real-time product browsing, cart management, and secure checkout.
Admin Dashboard: A custom-built ledger for administrators to track sales directly in the web app.
Automated Data Export: Structured SQL queries allow for seamless data loading into Power BI.
Responsive Design: Fully optimized for mobile and desktop viewing.
